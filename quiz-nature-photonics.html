How are these "most common words" counted?

1. The title data

  • arXiv physics.optics — every arXiv record whose categories field contains physics.optics (primary or cross-listed), from the Kaggle metadata dump (~56,600 titles).
  • Nature Photonics — every article pulled from the Crossref API by ISSN 1749-4893, 2007–2026 (~5,100 titles).
⚠️ The data was taken mid-2026, so 2026 is partial. Only years through 2025 are treated as complete, so the counts reflect titles as of 2025.

2. Turning a title into words

  1. Tokenize — lowercase; keep letters and inner hyphens, so single-photon stays one word.
  2. Lemmatize — a conservative singularizer maps plurals to one word (lasers→laser), with a protected set (physics, optics, …) and irregulars (indices→index).
  3. Drop stopwords — generic English and prose filler (using, novel, study, …) are removed.
  4. n-grams — adjacent uni-, bi-, and tri-grams are formed so phrases (frequency comb, perovskite solar cell) count as a unit.

3. Merging equivalent phrases

Each n-gram is reduced to a canonical key — the sorted set of its words — so counting ignores word order and hyphenation (machine-learning = machine learning; photonic integrated = integrated photonic). A few uninformative words (optical, light, …) are excluded.

4. Counting

  • Document frequency — each topic is counted once per title (presence, not raw frequency).
  • The quiz lists the 20 most frequent topics in each dataset, over titles through 2025.

5. Known limitations

  • The tokenizer is letters-only, so numerals and non-ASCII are stripped (MoS₂→mos, moiré→moir).
  • Lemmatization is rule-based, so a few forms split or merge imperfectly.
